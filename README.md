## LotationList [→Link](https://mrmocchy.github.io/LotationList/#/)
- 晩飯のメニューのルーティン管理用
- 一般に何周もするリストとして使用可能
- 保存はブラウザのローカルストレージなので別端末はおろか同端末の別ブラウザでもデータは共有されない
- 変則CSVでクリップボードにエクスポート＆インポートしてバックアップと共有ができる
  - （iOSでは1週間アクセスがなければローカルストレージが消去されるらしい）
### 操作方法
- 上下向きの<>で並び替え
- <ins>↓</ins>で一番下に移動
- 上部のゴミ箱ボタンで削除ボタン(赤)出現切り替え
- 各操作ごとに保存されるので保存ボタンはなし
- 上部の＋ボタンで追加画面に遷移
  - テキストボックスに入力してAddを押すとリスト最上部に追加
  - 下部のExportでリストを変則CSV（`,`ではなく`,\n`区切り）でクリップボードに保存
  - Importでクリップボードからリスト読み込み（リスト最下部に追加する形）
  - 各自メモなどに保存してバックアップをとること
